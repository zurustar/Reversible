# Personas — Reversible

アシッド系グルーヴボックスの想定ユーザ層(エレクトロニック/ダンスミュージック制作者、宅録〜ライブ)に合わせた2つのペルソナ。ストーリーはこの2名を主語に記述する。

---

## Persona 1: ケンジ(宅録トラックメイカー)
- **アーキタイプ**: 自宅でエレクトロニック/ダンスミュージック(テクノ、アシッドハウス、エレクトロ)を作る個人プロデューサー
- **技術習熟度**: 中〜高。DAWやシンセの基本は分かるが、重厚なDAWの立ち上げは面倒に感じることがある
- **利用文脈**: ヘッドホン/モニターで、ノートPCのブラウザ。空き時間にループを作って作り込む
- **ゴール / モチベーション**:
  - アシッドなベースラインとビートを素早くスケッチしたい
  - basslineのフィルターを回して「うねり」を追い込みたい
  - 作ったパターンを失わずに保存し、別マシンや友人と共有したい(JSON)
- **不満 / ペインポイント**:
  - 起動や設定が重いツール、音が途切れる(レイテンシ/グリッチ)
  - 作業内容が消える、保存フォーマットが閉じていて持ち出せない
- **成功の定義**: 「思いついてから数分でアシッドループが鳴り、保存・エクスポートまで完結する」
- **主に関わる機能**: bassline/ドラムの打ち込み、音作り、保存、JSONエクスポート/インポート

---

## Persona 2: アヤ(ライブ/ジャム志向パフォーマー)
- **アーキタイプ**: ライブやジャムでリアルタイムに操作して演奏する人。即興でパターンを切り替え、ノブを回して展開を作る
- **技術習熟度**: 高(特にリアルタイム操作)。安定性と即応性を最重視
- **利用文脈**: ライブ/配信/セッション。再生しながら手を動かし続ける
- **ゴール / モチベーション**:
  - 再生を止めずにフィルターやパラメータをリアルタイムに変化させたい
  - 音切れ(グリッチ)なく安定して鳴り続けてほしい
  - 事前に用意したパターン(JSON)を読み込んで即演奏に入りたい
- **不満 / ペインポイント**:
  - 演奏中のオーディオドロップアウト、UIの反応遅れ
  - 操作が深い階層に埋まっていて本番で触れない
- **成功の定義**: 「再生しっぱなしでノブを回しても破綻せず、思ったとおりに音が変化する」
- **主に関わる機能**: 安定したリアルタイム再生、テンポ、リアルタイムのパラメータ操作、パターン読み込み
